# Tentang projek ini 
Project game yang memiliki codename Ashes. Game ini dikembangkan untuk memenuhi penilaian semester akhir

# Deskripsi Game
Game dengan codename Ashes merupakan game horro yang bersifat linear. Pemain hanya menjalankan karakter utama dengan tujuan keluar dari tempat tersebut. 
Dengan beberapa ending yang berbeda dan cara yang berbeda pula untuk mendapatkan ending tersebut player dituntut agar dapat menyelesaikan puzzle dan mencari barang yang dibutuhkan untuk mendapatkan setiap ending.
